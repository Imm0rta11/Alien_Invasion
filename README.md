# Alien_Invasion
This game from book Python Crash Course.

For this game need Pygame. If you don't have it, install it using this command.
```
pip install pygame
```
or if you have Mac, Linux.
```
pip3 install pygame
```
# Launch game.
```
python main.py
```
or if you have Mac, Linux.

```
python3 main.py
```
Enjoy :)
